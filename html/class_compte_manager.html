Description détaillée

Classe qui s'occupe de la gestion des comptes.

Cette fonction gère entièrement les comptes à gérer par le logiciel et contient la racine de notre arborescence

Permet de creer un compte dans un dossier.

Paramètres
id_dossier: id du CompteVirtuel dans lequel créer un compte
nom: nom du compte à créer
type: type du compte, utilisation de la factory pour créer le compte à partir de son nom
virtuel: booleen permettant de choisir si le compte est virtuel (true) ou réel (false)
Renvoie
retourne true en cas de succès, else sinon

Effecue une recherche du "compte dossier" dans lequel créer un CompteVirtuel ou Compte,
si le "compte dossier" est trouvé, une vérification de son type est effectuée grâce à un cast.
le "Compte" est alors crée par la factory si il est réel.
