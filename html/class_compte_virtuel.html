Description détaillée

Dossier de Compte.

Un Comptevirtuel est un compte qui n’est destiné qu’à regrouper des sous-compte sans enregistrer d’opération

void CompteVirtuel::Add ( Compte *  file)
virtual

Permet d'ajouter un sous Compte.

Paramètres
file: pointeur vers le sous compte

L'ajout d'un fils se fait par vérification de son type,
si le fils est de même type que le dossier, alors il est ajouté au vecteur de compteabstrait du comptevirtuel. lors de l'ajout du fils,
le pointeur de père du fils est initialisé et un pointeur vers le fils est ajouté à m_listfichier.

Réimplémentée à partir de CompteAbstrait.

Permet de rechercher un compte parmis les fils d'un CompteVirtuel.

Paramètres
num: identifiant du sous compte

Rehderche un compte de manière récursive grâce à l'appel de la fonction GetCompte des sous CompteVirtuel si le compte recherché n'est pas trouvé dans m_listfichier.
Un cas particulier pour le "compte racine" étant donné que ce compte ne peut pas se trouver dans ses fils.

◆ getSolde()

double CompteVirtuel::getSolde ( ) const
virtual

Permet de connaitre la somme du solde des sous compte d'un CompteVirtuel.

Renvoie
retourne la valeur du solde calculé

Un Comptevirtuel ne possède pas d'attribut solde, son solde est calculé en faisant la somme de celui de ses fils,
pour chaque fils de type CompteVirtuel, le solde est recalculé de manière récursive.

Réimplémentée à partir de CompteAbstrait.
