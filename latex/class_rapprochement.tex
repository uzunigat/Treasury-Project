\doxysubsection{Description détaillée}
Le rapprochement d\textquotesingle{}un \mbox{\hyperlink{class_compte}{Compte}} consiste à figer des transactions passées. 

Cette classe permet de comparer les opérations saisies dans le logiciel avec les opérations effectuées réellement sur le compte bancaire
