\doxysubsection{Description détaillée}
Conteneur d\textquotesingle{}informations sur une opération concernant un \char`\"{}\+Compte réel\char`\"{}. 

Cette classe va nous permettre de stocker les informations relatives aux comptes prenants dans une transaction, notamment le débit, le crédit ainsi que le compte respectif auquel s\textquotesingle{}appliquera l\textquotesingle{}opération dans le contexte de la transaction
