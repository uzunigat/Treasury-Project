Description détaillée

Le rapprochement d'un Compte consiste à figer des transactions passées.

Cette classe permet de comparer les opérations saisies dans le logiciel avec les opérations effectuées réellement sur le compte bancaire

void Rapprochement::Afficher ( )

Afficher une Rapprochement Méthode qui permet d'afficher un raprochement.

Affiche l'ensemble des opérations concernant un rapprochement dans un compte,
si le pointeur de rapprochement dans une opération est le même que le reapprochement à afficher, alors on affiche l'opération.
